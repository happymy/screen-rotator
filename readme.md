# 屏幕旋转控制工具 (Screen Rotator)

一个基于 Python 的现代化屏幕旋转工具，专为 Windows 设计。  
提供直观的十字方向按钮、多显示器支持、定时自动旋转以及窗口置顶功能，所有设置自动保存。

---

## 功能特色

- ✅ **直观的方向控制**：使用上、右、下、左四个十字排列按钮，一键旋转屏幕。
- ✅ **多显示器支持**：可检测所有连接的显示器，选择任意一台进行旋转，名称友好显示。
- ✅ **自动旋转定时器**：可启用自动旋转，并自定义间隔秒数（1～86400秒）。
- ✅ **窗口置顶**：一键将控制窗口保持在其他窗口之上，方便操作。
- ✅ **配置自动记忆**：所有选项（显示器、方向、定时器、置顶状态）自动保存至用户目录，下次启动自动加载。
- ✅ **自适应界面**：采用可滚动框架，窗口缩小时自动出现垂直滚动条，完整显示所有控件。
- ✅ **现代暗色主题**：基于 `customtkinter` 构建，界面美观、支持高 DPI。

---

## 系统要求

- Windows 7 或更高版本（64位）
- **管理员权限**（旋转屏幕需要调用系统 API）
- 若打包为 EXE，无需安装 Python；若直接运行源码，需要 Python 3.8+

---

## 安装与运行

### 方式一：直接运行源码

1. 克隆或下载本项目。
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 以管理员身份运行：
   ```bash
   python screen_rotator.py
   ```

### 方式二：打包为独立 EXE

执行项目中的 `build.bat`（会自动创建虚拟环境并安装依赖）。

完成后在 `dist` 文件夹中找到 `ScreenRotator.exe`。

右键 → 以管理员身份运行。

## 配置文件

设置会自动保存到以下位置：

```text
%APPDATA%\ScreenRotator\config.json
```

包含内容：选择的显示器索引、旋转方向、定时器秒数、自动旋转开关状态、窗口置顶状态。
删除该文件可恢复默认设置。

## 依赖库

- `rotate-screen`：封装了 Windows ChangeDisplaySettingsEx API，用于获取和设置屏幕方向。
- `customtkinter`：提供现代化、可定制的 Tkinter 控件。

## 注意事项

- **必须管理员权限**：由于旋转屏幕需要调用系统级 API，没有管理员权限会导致操作失败。
- **多显示器环境**：如果显示器连接发生变化，请点击“刷新显示器列表”重新检测。
- **定时器限制**：为避免误操作，定时秒数限制在 1 秒至 86400 秒（24 小时）之间。
- **杀毒软件误报**：打包后的 EXE 可能会被部分杀软误报，请添加信任或使用代码签名证书。
